# Artifacts

Generated outputs from prompt-lab runs should be stored here.

Suggested artifact types:

- transcripts
- score reports
- prompt diffs
- rubric outputs
- aggregate comparison summaries

Portable reference assets also live here:

- `reference/prompt_eval_reference.sqlite3`
  - curated SQLite bundle for agent onboarding and cross-project reuse
- `reference/prompt_eval_reference_manifest.json`
  - on-disk manifest describing curation scope, schema, and entry points
- `reference/README.md`
  - human-readable orientation for the reference bundle
