# Vendable Documents & Templates

Reusable templates, boilerplate docs, and reference files that are useful across all builds.

Drop project-agnostic templates here so they can be copied into new projects as starting points.

## Examples of What Goes Here

- Project README templates
- Standard CONTRACT.md / ROADMAP.md skeletons
- CI/CD configuration templates
- Coding standard references
- Agent instruction templates
- Architecture decision record (ADR) templates

## Rule

Everything in this folder must be **project-agnostic**. If a document references a specific project name, path, or domain, it does not belong here.
